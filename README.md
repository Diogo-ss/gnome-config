# Script de pós instalção para Gnome 42.

``` shell
git clone https://github.com/Diogo-ss/gnome-config.git
```
``` shell
cd gnome-config && chmod +x setup.sh
```

Após executar o comando, instale a cosmic dock e ative a POP-Shell
